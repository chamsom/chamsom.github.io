Tic-Tac-Toe

Python

The program is built in Python in which the board is printed utilizing list comprehension. I also assigned the corresponding numbers to each tile on the board as a means of move validation for the player.

I am currently working on an iteration of the program that will implement an AI (computer) player that will pit itself against a human player.

To run the demo, in the shell of the Replit wesbite, type "python game.py".
