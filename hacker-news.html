Hacker News Clone

Vanilla JS & HTML/CSS

The project was finished in Summer 2020 as part of a capstone project for a JavaScript bootcamp and to further my studies in front-end web development. I attempted to emulate the well-known social news website, Hacker News.

Real-time data is requested from an endpoint provided by the Hacker News API. Link handling is done via the router.js library and reducers are utilized for state management.
